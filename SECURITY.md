# Security Policy

## Supported Versions

This project is currently maintained on the latest `main` branch.

## Reporting a Vulnerability

This repository is a small open-source project and is not expected to handle high-risk security scenarios, but responsible reports are welcome.

If you find a security-related problem:

1. Please do not open a public issue with sensitive details.
2. Use GitHub private reporting if available.
3. If private reporting is not available, open a minimal public issue without exploit details and mention that you have a security concern.

## Scope Notes

Please report issues such as:

- unsafe file write behavior
- path traversal risks
- accidental exposure of secrets or personal data
- unsafe network or API handling

General bugs, feature requests, and documentation issues should go through normal GitHub issues.
